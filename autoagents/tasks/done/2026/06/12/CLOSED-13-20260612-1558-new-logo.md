## Implementation summary
- Replaced KM0 pin SVG (removed shadow ellipse) in `host-www/opencloud-auth/favicon.svg` and `dex/web/themes/km0/favicon.svg`.
- Added `logo.svg` (same pin asset) under `host-www/opencloud-auth/` and `dex/web/themes/km0/`.
- Switched login landing (`login.html`), register page, Dex navbar (`header.html` via config), and LDAP card (`header-card.html`) from external `km0digital.com/brand/logo.png` to local `/logo.svg` / `theme/logo.svg`.
- Fixed Dex config key to `frontend.logoURL: theme/logo.svg` (Dex v2.42 ignores `logo:`).
- Added nginx `location = /logo.svg` alias in `nginx/snippets/opencloud-locations.conf`.
- Adjusted logo CSS sizing for square icon (96px max) on landing, register, and LDAP card views.

## Testing instructions

### Deploy
```bash
rsync -a /opt/opencloud/host-www/opencloud-auth/ /var/www/opencloud-auth/
sudo cp /opt/opencloud/nginx/snippets/opencloud-locations.conf /etc/nginx/snippets/opencloud-locations.conf
sudo nginx -t && sudo systemctl reload nginx
cd /opt/opencloud/dex && docker compose up -d --force-recreate dex
```

### 1. Public favicon and logo SVG
```bash
curl -sI https://cloud.km0digital.com/favicon.svg | head -3
curl -sI https://cloud.km0digital.com/logo.svg | head -3
curl -s https://cloud.km0digital.com/favicon.svg | grep -c ellipse
# expect HTTP 200, image/svg+xml, ellipse count 0 (new pin without shadow)
```

### 2. Login landing logo
```bash
curl -s https://cloud.km0digital.com/login.html | grep -F 'src="/logo.svg"'
# expect match; no km0digital.com/brand/logo.png
```

### 3. Register page logo
```bash
curl -s https://cloud.km0digital.com/register | grep -F 'src="/logo.svg"'
```

### 4. Dex navbar logo (connector picker)
```bash
curl -s "http://127.0.0.1:5556/dex/auth?client_id=OpenCloudWeb&redirect_uri=https://cloud.km0digital.com/oidc-callback.html&response_type=code&scope=openid" | grep 'theme/logo.svg'
```

### 5. Dex LDAP card logo
Open LDAP login from landing (*Sign in with username and password*) or inspect rendered HTML — card `<img class="km0-card__logo">` must reference `theme/logo.svg`.

### 6. OpenCloud SPA favicon override
```bash
curl -s https://cloud.km0digital.com/themes/opencloud/assets/favicon.svg | grep -F 'viewBox="0 0 32 32"'
curl -s https://cloud.km0digital.com/themes/opencloud/assets/favicon.svg | grep -c ellipse
# expect viewBox match, ellipse count 0
```

### 7. Visual check (browser)
- `/login.html` — gradient pin icon above “OpenCloud” heading.
- `/register` — same pin icon on registration card.
- Dex LDAP form — pin icon centered in card (no external wordmark PNG).
